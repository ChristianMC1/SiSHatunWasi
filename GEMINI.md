# Reglas de Desarrollo para SiSHatunWasi
- Siempre utiliza componentes de Flux UI para la interfaz (evita HTML nativo para tablas, modales e inputs).
- Las vistas de Livewire deben estar estructuradas usando Tailwind CSS.
- El idioma del código (variables y métodos) debe ser en inglés/español según las convenciones actuales.
